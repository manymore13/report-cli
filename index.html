💻 方式二 CLI 安装

1

安装

pip install report-cli
2

列出行业

report l
3

查询研报

report q -i 1046 -s 5

📖 CLI 命令参考

🎮 行业研报
report q -i 1046 -s 10
按行业代码查询，支持日期和页码筛选
📈 策略报告
report q -t strategy -s 5
市场策略与投资方向分析
🍷 个股研报
report q -t stock -c 600519
按股票代码精准查询，支持评级筛选
🌍 宏观研究
report q -t macro -s 5
宏观经济趋势与政策解读
🌅 券商晨报
report q -t morning -s 5
每日券商晨会观点汇总
📥 下载PDF
report d -i 1046 -s 5 -o ./reports
批量下载研报PDF，支持指定序号
